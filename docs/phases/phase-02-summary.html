src/desktop_agent/window.py
from PyQt6.QtWidgets import QWidget

class MainWindow(QWidget):
    def __init__(self):
        super().__init__()
        self.init_ui()

    def init_ui(self):
        self.setWindowTitle("Desktop Agent")
        self.setGeometry(100, 100, 800, 600)
        self.setMinimumSize(400, 300)
        self.setStyleSheet("background-color: #1a1a2e;")
src/desktop_agent/main.py
from PyQt6.QtWidgets import QApplication
from desktop_agent.window import MainWindow

def main():
    app = QApplication(sys.argv)  # Create app
    window = MainWindow()         # Create window
    window.show()                  # Show it
    return app.exec()              # Run event loop

Key Takeaways

1. Every PyQt App Needs QApplication

QApplication is the heart of your GUI application. It must be created before any widgets and manages the entire application lifecycle, settings, and event processing.

2. QWidget is the Base Class for Everything

All visible UI elements in Qt inherit from QWidget. A QWidget without a parent becomes a window. With a parent, it becomes a child widget inside that parent.

3. The Event Loop is Essential

app.exec() starts the event loop - a continuous cycle that waits for events (clicks, key presses, etc.) and dispatches them to widgets. Without it, your window would appear and immediately close.

4. Separate UI Setup into init_ui()

It's a common pattern to separate UI initialization into its own method. This keeps __init__ clean and makes the code more organized as your UI grows.

5. Geometry = Position + Size

setGeometry(x, y, width, height) sets both position and size in one call. For top-level windows, position is relative to the screen. You can also use move() and resize() separately.

"The event loop is like a patient waiter - always ready to respond, never blocking the restaurant."

The Pattern

Every PyQt application follows this fundamental pattern:

# 1. Import necessary classes
from PyQt6.QtWidgets import QApplication, QWidget

# 2. Create custom widget class
class MyWindow(QWidget):
    def __init__(self):
        super().__init__()
        # Setup UI here

# 3. Create app, create window, show window, run loop
app = QApplication(sys.argv)
window = MyWindow()
window.show()
app.exec()

This pattern will remain the same throughout our project. In future phases, we'll modify what happens inside MainWindow, but the overall structure stays consistent.
