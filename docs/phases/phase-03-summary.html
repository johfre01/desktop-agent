Design Decisions Made

Sharp Corners Instead of Rounded

We chose sharp corners to ensure precise screen capture - rounded corners would miss content in the corners of the capture area. This is a practical decision for a screenshot tool.

Thin 2px Border

A thinner border (2px instead of 3px) is less intrusive while still being clearly visible. It provides a good balance between visibility and minimizing visual obstruction.

Key Code Changes

# Window flags - frameless and always on top
self.setWindowFlags(
    Qt.WindowType.FramelessWindowHint |
    Qt.WindowType.WindowStaysOnTopHint
)

# Transparent background
self.setAttribute(Qt.WidgetAttribute.WA_TranslucentBackground)

# Custom paint event for the border
def paintEvent(self, event):
    painter = QPainter(self)
    painter.setRenderHint(QPainter.RenderHint.Antialiasing)

    pen = QPen(self.BORDER_COLOR)
    pen.setWidth(self.BORDER_WIDTH)
    painter.setPen(pen)
    painter.setBrush(Qt.BrushStyle.NoBrush)

    rect = self.rect().adjusted(...)
    painter.drawRect(rect)
    painter.end()

Key Takeaways

1. Window Flags Control Behavior

Qt window flags are powerful controls that determine how a window appears and behaves. Combining flags with | lets you create custom window types like our frameless, always-on-top overlay.

2. Transparency Requires Two Steps

Making a window transparent requires both removing the frame AND setting the translucent background attribute. Without both, you'll still see a solid window.

3. paintEvent is Your Canvas

Override paintEvent() to draw anything custom on your widget. Qt calls this automatically when the widget needs redrawing. Always create a QPainter, do your drawing, then end() it.

4. QPainter Uses Pen and Brush

QPen controls how lines/borders are drawn (color, width). QBrush controls how shapes are filled. Setting NoBrush keeps the interior transparent.

5. adjusted() Prevents Clipping

When drawing borders, use rect().adjusted() to shrink the drawing area slightly. This ensures thick borders don't get clipped at the window edges.

Current Limitation

  • No way to resize it
